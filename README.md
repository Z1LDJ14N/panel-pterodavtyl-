# 🎮 Pterodavtyl XYZ - Lightweight Server Panel

A lightweight Pterodactyl-like panel for managing Node.js 20 bots and scripts.

## Features

✅ **Lightweight & Fast** - Optimized for Vercel  
✅ **Node.js 20** - Full support  
✅ **Bot Management** - WhatsApp, Discord bots  
✅ **File Upload** - Upload & manage scripts  
✅ **Real-time Logs** - Monitor bot output  
✅ **Multiple Eggs** - Templates for quick setup  
✅ **User Authentication** - Secure login system
